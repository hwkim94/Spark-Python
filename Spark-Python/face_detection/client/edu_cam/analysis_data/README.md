## 집중도/참여도 분석을 위한 데이터

-파일명 별로
	- concentration : 정며에서 모니터를 바라보며 촬영
	- not_concentration : 정자세로 모니터를 쳐다보지만, 컴퓨터로 다른 작업을 하면서 촬영
	- glasses : 안경을 벗고 촬영(학습시에는 안경을 끼고 학습)
	- another person : 다른 인턴분의 얼굴로 촬영
	- sleep : 눈을 감고 정자세로 머리만 의자에 기대어 촬영
	- phone : 얼굴이 잘 인식되도록 모니터 높이에서 하면서 촬영
	- phone_under : 핸드폰을 키보드쪽에서 하면서 촬영

- 얼굴의 옆모습 혹은 엎드려있어서 얼굴이 보이지 않으면 인식을 하지 못함
